What am I looking at?

The plot treats the horizontal axis as time (from 0 to tau) and animates phase using the speed slider. These waveforms are the building blocks of shader math, shaping UVs, normals, or color ramps.

We start with a diagonal gradient and mix it with a rotated checkerboard. Warp introduces a sine-based offset, mimicking how shaders distort UVs for fire, water, or holograms.

How could I port this to GLSL?

Use the controls to read the current parameters and translate them to uniforms. Replace the canvas loop with a fragment shader loop over each pixel, keeping the same UV math: transform the UV vector, warp it with sin and cos, mix gradients, and finally output linear RGB.
